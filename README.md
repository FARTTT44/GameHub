# Pocket Arcade

Pocket Arcade is a browser‑based mini game hub featuring four unique games built with vanilla HTML, CSS and JavaScript.  Each game offers a different style of play while sharing a consistent neon‑arcade theme.  The site includes responsive design, dark/light modes, and interactive UI elements like a gallery slider and feedback form.

## Games

* **Reaction Click** – test your reflexes by clicking a moving target as many times as possible within the time limit.  Each click scores one point and the target jumps to a new position.
* **Rune Gambit** – a dice‑based combat game where you build up combos and acquire gold to improve your odds.
* **Neon Labyrinth** – navigate a procedurally generated maze while avoiding a hunter; grab keys and escape before you’re caught.
* **Glitch Garden** – a tower defence game with custom paths, animated towers, traps and varied enemy types.

## Features

* Responsive layout that scales across mobile, tablet and desktop.
* Dark and light themes with a retro neon aesthetic.
* Navigation bar to switch between sections.
* Animated image gallery and contact form with live feedback.
* Each game implements its own game logic in pure JavaScript; no external frameworks are used.

## How to Run

Open `index.html` in any modern browser.  The `style.css` and `script.js` files must reside alongside `index.html` for the site to work correctly.
