# Pocket Arcade Full Game Hub

A static HTML, CSS, and JavaScript browser game hub ready for GitHub Pages.

## Included files

- `index.html` - page structure and all game sections
- `style.css` - full responsive styling
- `script.js` - all game logic
- `.nojekyll` - keeps GitHub Pages from processing the site with Jekyll

## Included games

- Void Salvage
- Rune Gambit
- Neon Labyrinth
- Glitch Garden

## Upload to GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`, `style.css`, `script.js`, `README.md`, and `.nojekyll` to the repository root.
3. Open repository Settings.
4. Go to Pages.
5. Set Source to `Deploy from a branch`.
6. Choose the `main` branch and `/root`.
7. Save.
8. Open the GitHub Pages link once deployment finishes.

The files use relative links only, so the site works from the repository root without a build step.
