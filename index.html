Mini Game Hub

Four playable browser games in one responsive arcade hub: Void Salvage, Rune Gambit, Neon Labyrinth, and Glitch Garden.

Hub Rules

  1. Open the Games section.
  2. Pick a game tab.
  3. Use the start/reset buttons inside each game.
  4. Keyboard games use WASD or arrow keys.
  5. The hub is split into index.html, style.css, and script.js.

Featured Games

Void Salvage

Move a salvage ship, collect crystals, dodge mines, defeat drones, and buy upgrades.

Rune Gambit

Roll rune dice, lock symbols, build combos, block enemy attacks, and survive rounds.

Neon Labyrinth

Collect keys, use matching portals, avoid hazards, and outsmart hunters that scale over time.

Glitch Garden

Dig a clean endpoint-only path, place towers and traps, unlock areas, and fight the wave 100 boss.
